About FindAI

A free student project — runs locally or on Render cloud. No paid APIs. Checks photos, videos, audio, text, and documents for AI content.

{% if is_cloud %}

Running in cloud mode — models load on first scan.

{% endif %}

Architecture

  • app/engine/ — ML models, forensics, fusion, analyzers
  • app/core/ — scan pipeline + upload preview
  • app/db.py — SQLite history + analytics
  • FastAPI — web UI, REST API, Swagger docs at /docs

How it works

  1. You upload a file or paste text
  2. The app picks the right checker (image, audio, video, text, or document)
  3. Several free tools run — neural networks plus forensic checks
  4. You get one verdict with a confidence score and short explanation

Tools used (all free)

  • Images — Organika/sdxl-detector (ViT model) + edit forensics
  • Text — roberta-base-openai-detector + phrase analysis
  • Audio — librosa pitch analysis + splice detection
  • Video — frame-by-frame image model + temporal checks
  • Documents — PDF metadata + text model

Keep in mind

This gives probability estimates, not proof. Compressed WhatsApp forwards and very new AI models are harder to catch. Use results as a guide, not a final answer.
